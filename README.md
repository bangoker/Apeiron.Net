Apeiron.NET
===========

Combinaciones de Horarios para la Universidad de Guadalajara
##

Este es un projecto para leer información desde el SIIAU y generar horarios basados en condiciones como cupo, preferencia de maestro, etc.
En este momento se encuentra una librería/API usable, el plan es eventualmente agregar una aplicacion que use esta libreria.

Inspirado en el projecto original Apeiron http://sourceforge.net/projects/apeiron/ de Francisco Riestra.
